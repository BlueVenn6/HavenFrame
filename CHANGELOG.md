# Changelog / 更新记录

## 0.2.0-rc.12

- Published the bilingual HavenFrame Windows desktop source tree.
- Added independent image-generation and multimodal-extraction model routes.
- Added OpenAI native, OpenAI-compatible relay, Google Gemini image, Mainland China GLM, and international Z.AI GLM configuration paths.
- Added A4 board reports, structured procurement export, persistent assets, and installed-application release gates.
- Removed local-deployment, local-model-management, and local-renderer product UI from the public desktop product.

- 发布 HavenFrame 中英双语 Windows 桌面版源码。
- 独立图片生成与多模态提取模型线路。
- 支持 OpenAI 原生、OpenAI-compatible 中转、Google Gemini 图片、中国大陆 GLM 与国际 Z.AI GLM 配置。
- 增加 A4 方案板报告、结构化采购导出、资产持久化和安装版发布门禁。
- 从公开桌面产品中移除本地部署、本地模型管理和本地 renderer 产品 UI。
