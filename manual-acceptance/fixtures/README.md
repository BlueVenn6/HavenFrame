# 人工验收样例资料

这些文件只用于 Windows 人工验收环境，不代表 AI 已成功执行。

- `sample-project.json`：人工录入项目时的字段参考。
- `sample-requirements.txt`：单房间、多房间与空间渲染的需求文本。
- `sample-budget.csv`：结构化预算字段和中文编码对照样例，不是 AI 输出。
- `sample-room-plan.png`：可上传的中性平面/房间示意图，带有“非 AI 输出”标识。

涉及 OpenAI、Google 或 GLM 的步骤必须使用真实 Provider 和真实凭据。没有凭据时，在验收清单中记录 `BLOCKED_BY_CREDENTIALS`。
